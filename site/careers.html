Sr .Net SQL Developer

Full-Time
Remote (USA) 12+ Months 8+ Years Exp

About the Role

We are seeking a highly skilled Sr .Net SQL Developer to join our enterprise client team. You will design, develop, and maintain robust .NET applications and complex SQL Server databases, collaborating with cross-functional teams to deliver scalable business solutions.

Responsibilities

  • Design, develop, and maintain .NET applications (C#, ASP.NET, MVC, Web API).
  • Develop and optimize complex SQL Server queries, stored procedures, and triggers.
  • Collaborate with business analysts and stakeholders to gather requirements.
  • Perform code reviews, troubleshoot, and resolve technical issues.
  • Ensure best practices in security, scalability, and performance.

Requirements

  • Bachelor's degree in Computer Science or related field.
  • 8+ years of experience in .NET development (C#, ASP.NET, MVC, Web API).
  • Strong expertise in SQL Server (T-SQL, performance tuning, SSIS/SSRS a plus).
  • Experience with RESTful APIs, Entity Framework, and LINQ.
  • Excellent problem-solving and communication skills.
  • US work authorization required.

Nice to Have

  • Experience with Azure or AWS cloud platforms.
  • Agile/Scrum environment experience.
